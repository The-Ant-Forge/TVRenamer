Preferences

Open preferences via the Preferences menu or Ctrl+P.

File Handling

Destination Directory

Where renamed files are moved to. Leave empty to rename files in place.

  • Can be a local path or network share (UNC path like \\NAS\media)
  • Subdirectories are created automatically based on your rename format

Move Enabled

Whether to move files to the destination directory. If unchecked, files are renamed in their current location.

Rename Enabled

Whether to rename files. You can move files without renaming them by unchecking this.

Always Overwrite

When a file with the same name exists at the destination:

  • Unchecked - Create versioned copies: file (1).mkv, file (2).mkv
  • Checked - Overwrite existing files

Recursively Add Folders

When adding a folder, also scan its subfolders for video files.

Remove Emptied Directories

After moving files out of a folder, delete the folder if it is left empty.

Preserve Modification Time

Keep the file's original "last modified" timestamp when moving it, instead of the move time. Useful for sorting by date in file managers and media servers.

Delete Row After Move

Automatically remove a file's row from the table once it has been successfully processed. When unchecked, completed rows remain (with a green check icon) until you press Clear Completed.

Cleanup Duplicates

After moving a video file, optionally scan for other video files that represent the same episode:

  • Matches files with same base name or same season/episode numbers
  • Only deletes video files (preserves subtitle files)
  • Shows a confirmation dialog before deletion
  • Uses fuzzy matching (e.g., "S01E02" and "1x02" are recognized as the same episode)
  • Only triggers when moving video files — moving subtitles or metadata files does not trigger the duplicate scan

Rename Format

Rename Format String

Template for renamed filenames. See Renaming Files for available tokens.

Default: %S S%0sE%0e %t

Season Prefix

Text used for the per-season subdirectory name (e.g. Season produces Season 1, Season 2, …). Leave empty to put all episodes directly in the show's directory.

Drag and Drop Tokens

Drag tokens from the list into the format field to insert them.

Episode Ordering

Episode Ordering Preference

  • Aired - Original broadcast order (recommended for most uses)
  • DVD - DVD/BluRay release order (some shows reorder episodes)

Preload Folder

Enable Preload

Automatically scan a folder for video files when TVRenamer starts.

Preload Folder Path

The folder to scan. Useful for your download directory.

Metadata Tagging

Tag Video Files

Embed TV metadata directly into video files after renaming:

  • Show name, season, episode number, episode title, air date
  • Enables media players (Plex, Kodi, etc.) to display episode info

Supported Formats

Format Requirements
MP4, M4V, MOV Requires AtomicParsley or ffmpeg
MKV, WebM Requires MKVToolNix installed

See Metadata Tagging for more details.

Subtitles

Merge Sibling Subtitle Files

Automatically embed subtitle files (.srt, .ass, .ssa, .vtt) that sit next to a video file into the video container as soft subtitle tracks when the file is renamed:

  • MP4/M4V requires MP4Box (from GPAC)
  • MKV requires mkvmerge (from MKVToolNix)
  • Re-running a rename never adds duplicate language tracks

Default Language

The language assigned to a merged subtitle track when the subtitle's filename carries no language tag (e.g. show.s01e01.srt as opposed to show.s01e01.en.srt).

Delete Subtitle Files After Merge

Once a subtitle has been successfully embedded, delete the standalone subtitle file. Leave unchecked to keep both.

See Subtitle Merging for full details, including how subtitles are paired with episodes and per-file progress.

Updates

Check for Updates

On startup, check whether a newer TVRenamer release is available and show a notification link if so. No update is ever installed automatically.

Show Matching

Show Overrides

Map extracted show names to different search terms. Useful when:

  • The show name in your filenames differs from TheTVDB's name
  • You want to force a specific spelling or variation

Quick add: When a show can't be found, click the "click to set Hint" link in the Proposed File Path column to open Preferences directly on the Matching tab with the show name pre-filled.

See Show Matching for details.

Remembered Disambiguations

When multiple shows match a query, TVRenamer prompts you to choose. Your selection is remembered for future lookups.

Appearance

Theme

  • Light - Light background, dark text
  • Dark - Dark background, light text
  • Auto - Follow system setting
Note: Theme changes require restarting TVRenamer to take effect.

Ignore Keywords

Comma-separated list of keywords. Files whose path or name contains any of these words will be skipped entirely and not renamed. Matching is case-insensitive.

Example: sample,proof,extras would skip files like "show.s01e01.sample.mkv" or "show.s01e01.extras.mkv".

The default value is sample, which filters out sample/preview files.
